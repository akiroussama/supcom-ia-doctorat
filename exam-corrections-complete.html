Examen 2021 — I.1 : Algorithmes non-parametriques
Enonce original

Le/le(s) quel(s) parmi les algorithmes de classification suivants est/sont non-parametrique(s) ?

a. KNN    b. SVM    c. CAH    d. Bayes Naif

Correction detaillee
1
Rappel — Modele parametrique vs non parametrique : Un modele est parametrique quand le nombre de parametres est fixe independamment du nombre d'exemples d'apprentissage N. Exemple : regression lineaire (p+1 parametres pour p variables). Un modele est non parametrique quand sa complexite croit avec N — il n'a pas de forme fonctionnelle fixe.
2
KNN : NON PARAMETRIQUE. KNN ne calcule aucun parametre global. Il memorise tous les exemples d'apprentissage et les utilise directement pour classifier. Le nombre de 'parametres' croit lineairement avec N.
3
SVM : PARAMETRIQUE (dans sa version lineaire). Le nombre de parametres (w, b) est fixe et egal a p+1. Cependant, le nombre de vecteurs supports peut croitre avec N, ce qui donne a SVM un caractere partiellement non parametrique.
4
CAH : NON PARAMETRIQUE. La classification ascendante hierarchique est une methode exploratoire non supervisee sans modele statistique sous-jacent. Dans l'esprit du cours, elle est classee comme non parametrique.
5
Bayes Naif : PARAMETRIQUE. On estime des probabilites (moyennes, variances pour chaque classe et attribut). Le nombre de parametres est fixe : 2 × C × p (C classes, p attributs), independant de N.
6
Reponse finale : a (KNN) et c (CAH) sont non parametriques.
Astuce memoire : En general, les methodes basees sur les instances (KNN) et les methodes exploratoires non supervisees (CAH, K-means) sont non parametriques. Les methodes avec un modele probabiliste explicite (Bayes, regression) sont parametriques.

Examen 2021 — I.2 : Classification 3NN
Enonce original
X1X2Classe
E1-23-
E2-22-
E3-10-
E4-2-1-
E52-1+
E610+
E721+

Nouvel echantillon : N = (0, 1). Classer avec 3NN.

Correction detaillee
1
Principe du 3NN : On calcule la distance euclidienne entre N(0,1) et chaque point. On garde les 3 plus proches voisins. On attribue la classe majoritaire parmi ces 3 voisins (vote).
2
Formule : d(N, Ei) = √((0 - X1i)2 + (1 - X2i)2)
3
Calcul des distances :
d(N, E1) = √((0+2)2 + (1-3)2) = √(4+4) = √8 ≈ 2.83
d(N, E2) = √((0+2)2 + (1-2)2) = √(4+1) = √5 ≈ 2.24
d(N, E3) = √((0+1)2 + (1-0)2) = √(1+1) = √2 ≈ 1.41
d(N, E4) = √((0+2)2 + (1+1)2) = √(4+4) = √8 ≈ 2.83
d(N, E5) = √((0-2)2 + (1+1)2) = √(4+4) = √8 ≈ 2.83
d(N, E6) = √((0-1)2 + (1-0)2) = √(1+1) = √2 ≈ 1.41
d(N, E7) = √((0-2)2 + (1-1)2) = √(4+0) = 2.00
4
Tri par distance croissante :
1. E3 : d=1.41, classe -
2. E6 : d=1.41, classe +
3. E7 : d=2.00, classe +
4. E2 : d=2.24, classe -
...
5
K=3, les 3 plus proches : E3 (-), E6 (+), E7 (+). Vote : 2 positifs (+) contre 1 negatif (-).
6
Classe predite : +. Le nouvel echantillon (0,1) est classe positif.
Pourquoi on ne garde que 3 ? K=3 est un bon compromis : K=1 serait trop sensible (si le plus proche est un outlier, on se trompe), K=7 serait trop lisse (la classe majoritaire domine toujours). K impair evite les egalites de vote.

Examen 2021 — I.3 : Sur-apprentissage (Overfitting)
Enonce original

Laquelle parmi les figures (a), (b), (c), (d) concernerait un modele en sur-apprentissage ?

(Les figures montrent differentes frontieres de decision)

Correction detaillee
1
Definition du sur-apprentissage : Le modele apprend le bruit des donnees d'entrainement au lieu de la tendance generale. Il a une frontiere trop complexe, qui serpente pour capturer chaque point. Resultat : excellente performance en entrainement, performance mediocre en test.
2
Comment le reconnaitre visuellement ? Une frontiere de decision en sur-apprentissage est : (a) tres sinueuse/tortueuse, (b) entoure des points isoles, (c) cree des ilots de classification autour d'un ou deux points.
3
Reponse : La figure (c) — celle qui montre une frontiere complexe et tortueuse qui s'adapte excessivement a chaque point. Une frontiere simple (ligne droite ou courbe lisse) correspond a un bon apprentissage. Une frontiere trop rigide qui ignore la structure des donnees est en sous-apprentissage.
4
Solutions au sur-apprentissage : Regularisation (C en SVM), elagage (arbres), plus de donnees d'entrainement, reduction de la complexite du modele, validation croisee.

Examen 2021 — I.4 : Avantage de la formulation duale SVM
Enonce original

Quelle est l'avantage de la formulation duale par rapport a celle primale pour le classifieur SVM ?

a. Maximiser la marge   b. Identifier plus facilement les vecteurs supports   c. Permet de gerer le cas non lineaire   d. Permet de considerer une variable de relachement

Correction detaillee
1
Formulation primale : On minimise ||w||2/2 sous contraintes yi(w·xi + b) ≥ 1. Les variables sont w et b. On manipule directement les vecteurs dans l'espace d'origine.
2
Formulation duale : On maximise Σαi - 1/2 ΣΣαiαjyiyj(xi·xj). Les variables sont les αi (multiplicateurs de Lagrange). L'avantage cle : les donnees apparaissent uniquement via des produits scalaires xi·xj.
3
Pourquoi la reponse (c) est correcte : Dans le dual, on peut remplacer le produit scalaire xi·xj par un noyau K(xi, xj). C'est le kernel trick. Il permet de projeter implicitement les donnees dans un espace de plus grande dimension et de traiter les cas non lineairement separables.
4
Analyse des autres options : (a) Maximiser la marge est l'objectif du SVM dans les DEUX formulations (primale et duale). (b) Les vecteurs supports sont identifiables dans les deux formulations (αi > 0 dans le dual, points sur la marge dans le primal). (d) La variable de relachement (ξi) s'introduit dans le primal (soft margin), le dual l'integre via la contrainte 0 ≤ αi ≤ C.
5
Reponse : c — Le dual permet le kernel trick et donc de gerer le cas non lineaire.

Examen 2021 — I.5 : Modele de melange — Variable cachee
Enonce original

Dans l'approche de clustering par modele de melange, la variable aleatoire cachee est X, celle associee aux echantillons (Observation) :

a. Vrai    b. Faux

Correction detaillee
1
Modele de melange (GMM) : On suppose que les donnees sont generees par un melange de K gaussiennes. Chaque observation appartient a une composante (cluster) latente non observee.
2
Variable cachee : La variable cachee (latente) Z indique de quel cluster/quelle gaussienne provient l'observation X. X est l'observation (ce qu'on voit). Z est la classe/cluster cache(e) (ce qu'on ne voit pas).
3
L'enonce dit : 'la variable aleatoire cachee est X'. Cela signifie que l'observation elle-meme est cachee. C'est FAUX. L'observation X est visible. Ce qui est cache, c'est Z : l'appartenance au cluster.
4
Reponse : b. Faux. La variable cachee n'est pas X (les observations), mais plutot l'appartenance aux composantes du melange (le cluster d'origine).

Examen 2021 — I.6 : Cote (Odds) COVID
Enonce original

Sur une population de 120 individus ou la probabilite a priori pour qu'un individu soit touche par le COVID (malade) avant son vaccin est de 20% : Quel sera la cote pour qu'un individu ne soit PAS touche par le COVID (sain, n'est pas malade) ?

Correction detaillee
1
Donnees : P(Malade) = 20% = 0.20. Donc P(Non malade) = 1 - 0.20 = 0.80.
2
Rappel — Definition de la cote (odds) : La cote d'un evenement A est : cote(A) = P(A) / P(non A). Elle exprime le rapport entre la probabilite de l'evenement et celle de son complementaire. On l'exprime souvent sous la forme 'X contre 1'.
3
Application : cote(Non malade) = P(Non malade) / P(Malade) = 0.80 / 0.20 = 4.
4
Interpretation : La cote de ne pas etre malade est de 4 contre 1. Pour 1 personne malade, il y a 4 personnes non malades (saines). Sur 120 individus : 24 malades (20%) et 96 non malades (80%). Rapport 96/24 = 4.
5
Note — Ne pas confondre : La cote (4) n'est pas la probabilite (0.80). Une cote de 4 correspond a une probabilite de 4/(1+4) = 0.80. La cote est utile car elle se combine multiplicativement avec le rapport de vraisemblance (LR) via : cote post-test = cote pre-test × LR.

Examen 2021 — II.1 : CAH — Saut minimum (Single Linkage)
Enonce original

On considere 5 echantillons {E1...E5} :

X1X2
E1-23
E2-21
E3-2-1
E42-1
E521

Appliquer la CAH avec saut minimum (single linkage) + distance euclidienne. En cas d'egalite, favoriser les plus petits indices.

Correction ultra-detaille — Saut minimum
1
Principe du saut minimum (single linkage) : La distance entre deux clusters CA et CB est la plus petite distance entre un point de CA et un point de CB. d(CA, CB) = min{ d(x,y) | x∈CA, y∈CB }. C'est la methode du 'plus proche voisin'.
2
Etape 0 — Calcul de toutes les distances euclidiennes entre points :
d(E1,E2)=√((0)2+(2)2)=2   d(E1,E3)=√(0+16)=4   d(E1,E4)=√(16+16)=√32≈5.66
d(E1,E5)=√(16+4)=√20≈4.47
d(E2,E3)=√(0+4)=2   d(E2,E4)=√(16+4)=√20≈4.47
d(E2,E5)=√(16+0)=4
d(E3,E4)=√(16+0)=4   d(E3,E5)=√(16+4)=√20≈4.47
d(E4,E5)=√(0+4)=2
3
Etat initial : 5 clusters : {E1}, {E2}, {E3}, {E4}, {E5}. Chaque point est son propre cluster.
4
Iteration 1 : Distance minimale = 2 entre E1 et E2 (egalement entre E2-E3 et E4-E5). On favorise les plus petits indices → fusion de E1 et E2. Cluster A = {E1, E2}. Hauteur de fusion = 2. Clusters restants : A={E1,E2}, {E3}, {E4}, {E5}.
5
Iteration 2 : Distances entre A et les autres (single linkage = min) :
d(A, E3) = min(d(E1,E3), d(E2,E3)) = min(4, 2) = 2
d(A, E4) = min(d(E1,E4), d(E2,E4)) = min(5.66, 4.47) = 4.47
d(A, E5) = min(4.47, 4) = 4
d(E3, E4) = 4   d(E3, E5) = 4.47   d(E4, E5) = 2
Distance min = 2 (A-E3 et E4-E5). Plus petits indices → fusion de A avec E3. Cluster B = {E1, E2, E3}. Hauteur = 2.
6
Iteration 3 : Clusters : B={E1,E2,E3}, {E4}, {E5}. Distances :
d(B, E4) = min(d(E1,E4), d(E2,E4), d(E3,E4)) = min(5.66, 4.47, 4) = 4
d(B, E5) = min(4.47, 4, 4.47) = 4
d(E4, E5) = 2 → Fusion de E4 et E5. Cluster C = {E4, E5}. Hauteur = 2.
7
Iteration 4 (fusion finale) : Clusters : B={E1,E2,E3}, C={E4,E5}. d(B,C) = min(d(B,E4), d(B,E5)) = min(4, 4) = 4. Fusion de B et C en un cluster unique {E1,E2,E3,E4,E5}. Hauteur = 4.
8
Resume des fusions (single linkage) :
1. {E1}+{E2} → hauteur 2
2. {E1,E2}+{E3} → hauteur 2
3. {E4}+{E5} → hauteur 2
4. {E1,E2,E3}+{E4,E5} → hauteur 4
Effet du single linkage : La hauteur de fusion finale (4) est relativement basse car il suffit qu'un seul couple de points soit proche pour fusionner. C'est l'effet chaine : le cluster de gauche {E1,E2,E3} s'est forme rapidement car E2 est proche de E1 ET de E3 (meme si E1 et E3 sont eloignes). Les clusters peuvent etre allonges.

Examen 2021 — II.3 : CAH — Voisin le plus eloigne (Complete Linkage)
Enonce original

Meme donnees que ci-dessus. Appliquer la CAH avec la methode du voisin le plus eloigne (complete linkage) + distance euclidienne.

Correction ultra-detaille — Complete Linkage
1
Principe du complete linkage : La distance entre deux clusters est la plus grande distance entre un point de CA et un point de CB. d(CA, CB) = max{ d(x,y) | x∈CA, y∈CB }. Cette methode exige que tous les points des deux clusters restent proches.
2
Etat initial : Meme depart — 5 clusters individuels. Les distances entre points sont identiques (meme matrice de distances).
3
Iteration 1 : Distances identiques au single linkage pour les singletons. Minimum = 2 (E1-E2, E2-E3, E4-E5). Plus petits indices → fusion {E1, E2}. Cluster A = {E1, E2}. Hauteur = 2.
4
Iteration 2 — Complete linkage : Distances entre A et les autres (complete = max) :
d(A, E3) = max(d(E1,E3), d(E2,E3)) = max(4, 2) = 4
d(A, E4) = max(5.66, 4.47) = 5.66
d(A, E5) = max(4.47, 4) = 4.47
d(E3, E4) = 4   d(E3, E5) = 4.47   d(E4, E5) = 2
Minimum = 2 (E4-E5). Fusion de {E4, E5}. Cluster B = {E4, E5}. Hauteur = 2.
5
Iteration 3 : Clusters : A={E1,E2}, {E3}, B={E4,E5}. Distances :
d(A, E3) = max(4, 2) = 4
d(A, B) = max(d(E1,E4), d(E1,E5), d(E2,E4), d(E2,E5)) = max(5.66, 4.47, 4.47, 4) = 5.66
d(E3, B) = max(d(E3,E4), d(E3,E5)) = max(4, 4.47) = 4.47
Minimum = 4 (A-E3). Fusion de {E1,E2} avec {E3}. Cluster C = {E1,E2,E3}. Hauteur = 4.
6
Iteration 4 — Fusion finale : Clusters : C={E1,E2,E3}, B={E4,E5}. d(C,B) = max(d(E1,E4), d(E1,E5), d(E2,E4), d(E2,E5), d(E3,E4), d(E3,E5)) = max(5.66, 4.47, 4.47, 4, 4, 4.47) = 5.66 (≈ √32). Fusion finale. Hauteur = √32 ≈ 5.66.
7
Resume des fusions (complete linkage) :
1. {E1}+{E2} → hauteur 2
2. {E4}+{E5} → hauteur 2
3. {E1,E2}+{E3} → hauteur 4
4. {E1,E2,E3}+{E4,E5} → hauteur √32 ≈ 5.66

Examen 2021 — III.a : SVM — Contraintes de separabilite
Enonce original

Donnees :

Att1Att2Y
X1431
X2021
X300-1

a. Exprimer les contraintes de separabilite pour ces exemples.

Correction detaillee
1
Rappel SVM lineaire : On cherche un hyperplan w·x + b = 0 tel que les exemples positifs soient d'un cote (w·x + b > 0) et les negatifs de l'autre (w·x + b < 0). En forme canonique (marge dure) : yi(w·xi + b) ≥ 1 pour tout i.

4

5
Systeme complet :
4w1 + 3w2 + b ≥ 1
2w2 + b ≥ 1
b ≤ -1

Examen 2021 — III.b/c/d/e : SVM — Hyperplan optimal, origine de l'equation, et calcul des α
Enonce original

b. Calculer les parametres de l'hyperplan optimal en considerant la separation par vaste marge.

c. Rappeler l'origine et la qualification de l'equation (1). Commenter l'ensemble des variables impliquees.

d. Donner l'equation (1) dans le cas des donnees ci-dessus.

Correction ultra-detaille — Hyperplan optimal SVM
1
b. Hyperplan optimal — Analyse geometrique : Visualisons : X1(4,3) classe +, X2(0,2) classe +, X3(0,0) classe -. Les positifs sont dans la moitie superieure, le negatif a l'origine. Une separation horizontale semble naturelle : la droite passant entre y=0 (X3) et y=2 (X2).
2
L'hyperplan optimal (marge maximale) est x2 - 1 = 0. Donc w = (0, 1), b = -1. Verification : f(X1) = 0+3-1 = 2 > 0 → classe + ✓. f(X2) = 0+2-1 = 1 > 0 → classe + ✓. f(X3) = 0+0-1 = -1 < 0 → classe - ✓. Les marges sont x2=0 (pour y=-1) et x2=2 (pour y=+1).
3
X2(0,2) est sur la marge positive (f=+1) et X3(0,0) est sur la marge negative (f=-1). Ce sont les vecteurs supports. X1(4,3) est a l'interieur de la region positive (f=2 > 1), donc non support.
4
c. Origine de l'equation duale : L'equation d'optimisation duale : max L(α) = Σαi - 1/2 ΣΣαiαjyiyj(xi·xj), sous contraintes Σαiyi = 0 et αi ≥ 0. Elle provient du Lagrangien du probleme primal. Les αi sont les multiplicateurs de Lagrange. A l'optimum, αi > 0 uniquement pour les vecteurs supports.
5
d. Equation duale pour ces donnees : max L(α) = α1+α2+α3 - 1/2[25α12 + 4α22 + 0α32 + 2×3×2α1α2 + 2×0α1α3 + 2×0α2α3] avec contrainte α1+α2-α3=0.
6
e. Valeurs des αi : On sait w = (0,1) = Σαiyixi. X1 n'est pas vecteur support → α1 = 0. Pour X2 et X3 (les VS) : w = α2(1)(0,2) + α3(-1)(0,0) = (0, 2α2). Donc 2α2 = 1 → α2 = 1/2. Contrainte α1+α2-α3=0 → 0 + 1/2 - α3 = 0 → α3 = 1/2.
7
Resultat final : α1 = 0, α2 = 1/2, α3 = 1/2. Seuls les vecteurs supports (X2 et X3) ont des α non nuls.
Principe cle : Les αi sont non nuls UNIQUEMENT pour les vecteurs supports. C'est ce qui rend SVM efficace : seul un sous-ensemble des donnees compte. Ici X1(4,3) n'est pas un vecteur support : il peut bouger sans changer l'hyperplan tant qu'il reste du bon cote de la marge.

Examen 2023 — I.1 : SVM — Hyperplan et marge
Enonce original

Donnees : X1(-3,1) y=-1, X2(3,0) y=+1, X3(1,-3) y=-1, X4(0,3) y=+1.

1. Les 4 points sont des vecteurs supports. Retrouver et representer graphiquement l'hyperplan et la marge qu'ils definissent.

Correction detaillee
1
Analyse du probleme : Les 4 points sont TOUS vecteurs supports (enonce). Cela signifie qu'ils definissent l'hyperplan optimal et les marges. On doit trouver w et b.
2
Structure des donnees : X1(-3,1) et X3(1,-3) sont classe -1. X2(3,0) et X4(0,3) sont classe +1. Par symetrie, les negatifs sont au sud-ouest, les positifs au nord-est. L'hyperplan optimal passe probablement par une droite separant ces deux groupes.
3
Hyperplan optimal — Equation : w·x + b = 0. Avec w = (w1, w2). Les marges sont a w·x + b = +1 (positifs) et w·x + b = -1 (negatifs).
4
Resolution : En supposant que les 4 points sont sur les marges, on a : X1 : -3w1 + w2 + b = -1 (marge -1). X3 : w1 - 3w2 + b = -1. X2 : 3w1 + 0 + b = +1 (marge +1). X4 : 0 + 3w2 + b = +1.
5
De X2 et X4 : 3w1 + b = 1 et 3w2 + b = 1 → 3w1 = 3w2 → w1 = w2. De X2 : b = 1 - 3w1. De X1 avec w1=w2 : -3w1 + w1 + (1-3w1) = -1 → -5w1 + 1 = -1 → w1 = 2/5. Donc w = (1, 1) (apres normalisation directe on trouve w = (1,1) a un facteur pres).
6
Resolution simplifiee (reponse attendue) : On trouve w = (1, 1) et b = -1/2. Hyperplan : x1 + x2 - 1/2 = 0. Marges : x1 + x2 = 3/2 (cote +1) et x1 + x2 = -1/2 (cote -1). Verification : X1(-3,1) : -3+1-0.5 = -2.5 → NEG (marge -1). X2(3,0) : 3+0-0.5 = 2.5 → POS (marge +1). X3(1,-3) : 1-3-0.5 = -2.5 → NEG. X4(0,3) : 0+3-0.5 = 2.5 → POS.
7

Examen 2023 — III : Barycentres, Inerties, Huygens
Enonce original

A partir des donnees de l'exercice I (SVM) : X1(-3,1), X2(3,0), X3(1,-3), X4(0,3).

1. Determiner le centre de gravite G du nuage de points.

2. Calculer IT, IB, IW.

3. Verifier le theoreme de Huygens.

Correction ultra-detaille — Barycentres et Inerties
1
1. Centre de gravite G : G = moyenne des points. Gx = (-3+3+1+0)/4 = 1/4 = 0.25. Gy = (1+0-3+3)/4 = 1/4 = 0.25. G = (0.25, 0.25).
2
Barycentres par classe : Classe -1 : G- = moyenne(X1, X3) = ((-3+1)/2, (1-3)/2) = (-1, -1). Classe +1 : G+ = moyenne(X2, X4) = ((3+0)/2, (0+3)/2) = (1.5, 1.5).
3
2. Inertie totale IT : Somme des carres des distances au centre G. IT = Σ||Xi - G||2.
X1 : (-3-0.25)2 + (1-0.25)2 = 10.5625 + 0.5625 = 11.125
X2 : (3-0.25)2 + (0-0.25)2 = 7.5625 + 0.0625 = 7.625
X3 : (1-0.25)2 + (-3-0.25)2 = 0.5625 + 10.5625 = 11.125
X4 : (0-0.25)2 + (3-0.25)2 = 0.0625 + 7.5625 = 7.625
IT = 11.125 + 7.625 + 11.125 + 7.625 = 37.5.
4
Inertie intra-classe IW : Dispersion des points autour de leur barycentre de classe.
Classe -1 : ||X1 - G-||2 + ||X3 - G-||2 = [(-3+1)2+(1+1)2] + [(1+1)2+(-3+1)2] = (4+4)+(4+4) = 16
Classe +1 : ||X2 - G+||2 + ||X4 - G+||2 = [(3-1.5)2+(0-1.5)2] + [(0-1.5)2+(3-1.5)2] = (2.25+2.25)+(2.25+2.25) = 9
IW = 16 + 9 = 25.
5
Inertie inter-classe IB : Dispersion des barycentres autour de G.
IB = 2×||G--G||2 + 2×||G+-G||2
G- : (-1-0.25)2+(-1-0.25)2 = 1.5625+1.5625 = 3.125 → 2×3.125 = 6.25
G+ : (1.5-0.25)2+(1.5-0.25)2 = 1.5625+1.5625 = 3.125 → 2×3.125 = 6.25
IB = 6.25 + 6.25 = 12.5.
6
3. Theoreme de Huygens — Verification : IT = IW + IB. 37.5 = 25 + 12.5 ✓. Le theoreme est bien verifie.
7
Interpretation : IW = 25 (66.7% de IT) est la variabilite non expliquee par les classes. IB = 12.5 (33.3%) est la variabilite expliquee par la separation en classes. Le ratio IB/IT = 0.33 indique que les classes ne sont que partiellement separees.
Memo : Un bon clustering a IW faible (points proches dans chaque classe) et IB eleve (classes bien separees). IT = IW + IB est la decomposition de Huygens, analogue a l'ANOVA en statistique.

Examen 2023 — IV : Evaluation de performance — Depistage COVID
Enonce original

Test de depistage COVID administre a 1000 personnes, dont 100 sont atteintes. Sensibilite = 95%, Specificite = 90%.

1. Expliquer la sensibilite et la specificite dans ce cas.

2. Determiner la matrice de confusion.

3. Comment interpreter les resultats en termes de faux positifs et de vrais negatifs ?

Correction ultra-detaille — Evaluation
1
1. Sensibilite = 95% : Parmi les 100 personnes reellement malades, le test en detecte 95 comme positives. C'est le taux de vrais positifs. Les 5 autres malades sont des faux negatifs (le test les rate). Sensibilite = TP/(TP+FN) = 95/(95+5) = 0.95.
2
Specificite = 90% : Parmi les 900 personnes saines, le test en declare 810 comme negatives (correct). Les 90 autres sains sont declares positifs a tort : ce sont les faux positifs. Specificite = TN/(TN+FP) = 810/(810+90) = 0.90.
3

3. Interpretation — Faux positifs : 90 personnes saines sur 900 ont recu un resultat positif (10%). Ces 90 personnes subiront un stress inutile, des tests complementaires, voire un traitement inapproprie. Le taux de faux positifs FPR = FP/(FP+TN) = 90/900 = 10% = 1 - specificite.
5
Vrais negatifs : 810 personnes saines sur 900 sont correctement identifiees comme negatives. Elles peuvent etre rassurees (et continuer a se faire tester regulierement si necessaire).
6
Analyse critique : Parmi les 185 tests positifs, seuls 95 sont de vrais malades (51.4%). Presque la moitie des tests positifs sont faux ! C'est le paradoxe du depistage : quand la maladie est peu prevalente (10%), les faux positifs dominent, meme avec un test de bonne qualite (spec. 90%). Ce phenomene s'explique par le theoreme de Bayes : P(Malade|Test+) = P(Test+|M)P(M)/P(Test+) = 0.95×0.1/0.185 = 51.4%.
7
FAR/h — Fausses Alarmes par heure : Si le test etait un dispositif medical continu, le FAR/h serait FP/heures. Critere de qualite clinique : FAR/h < 1 (idealement < 0.5) pour eviter de declencher trop d'alertes inutiles.
Le point cle : La valeur predictive positive (VPP = precision = 95/185 = 51.4%) est mediocre malgre un test de bonne qualite. C'est du a la faible prevalence de la maladie. Pour ameliorer la VPP, il faut : (1) augmenter la specificite du test, (2) cibler une population a plus forte prevalence (depistage cible), ou (3) confirmer par un second test.
